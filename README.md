Magic Dust - Manual de Usuario
1. Introducción

Magic Dust es un editor de texto multiplataforma creado en Python con Tkinter.
Permite abrir, editar, guardar, buscar texto y gestionar archivos de distintos formatos.

2. Requisitos

Sistema operativo: Windows, Linux, macOS

Python 3.x instalado

Librerías: Tkinter (incluida en Python por defecto)

3. Instalación

Descargar el archivo MagicDustApp.py y el icono icono.ico.

Guardar todos los archivos en la misma carpeta.

Abrir la terminal o consola y ejecutar:

python MagicDustApp.py


La aplicación se abrirá en una ventana de 800x550 píxeles.

4. Interfaz de usuario

Área de texto: Para escribir y editar contenido.

Barra de estado: Muestra el número de caracteres del archivo.

Menú principal: Contiene las opciones Archivo, Editar y Ayuda.

5. Menú Archivo

Abrir (Ctrl + O): Selecciona un archivo existente para editar.

Guardar (Ctrl + S): Guarda los cambios en el archivo abierto.

Guardar como: Guarda el contenido en un nuevo archivo.

Salir: Cierra la aplicación.

6. Menú Editar

Deshacer: Revierte la última acción.

Rehacer: Reaplica la última acción deshecha.

Buscar (Ctrl + F): Busca palabras o frases dentro del texto y las resalta.

Copiar / Cortar / Pegar: Funciones de edición básicas.

7. Menú Ayuda

Información: Muestra información de la aplicación.

Manual de usuario: Abre este manual en PDF o web.

Integrantes: Muestra los miembros del grupo de desarrollo.

8. Funcionalidades adicionales

Soporte de archivos: .txt, .py, .cpp, .cs, y otros.

Resalta todas las coincidencias al buscar palabras.

Interfaz limpia y fácil de usar.

9. Consejos de uso

Guardar con frecuencia para evitar pérdida de información.

Para buscar palabras similares, usar la función de búsqueda repetidamente.

Mantener todos los archivos de la aplicación en la misma carpeta para evitar errores de ruta.
